Set Up Your Environment

Install the Stacks CLI and set up your development environment with our comprehensive toolkit.

2

Write Your First Contract

Learn Clarity and write your first smart contract with our interactive tutorials and examples.

3

Test & Deploy

Test your contracts locally and deploy to testnet or mainnet with confidence.

clarinet test

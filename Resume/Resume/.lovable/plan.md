
# Resume Website for Dhruv Ozha

## Overview
A professional, ATS-friendly, single-page resume website with a modern blue color scheme, clean typography, and a PDF download feature. The design will be static and recruiter-friendly.

## Design Approach
- **Color Palette**: Modern blue accents (#2563EB / navy blue) on a clean white background
- **Typography**: Professional sans-serif fonts for readability
- **Layout**: Single-column, top-to-bottom flow for maximum ATS compatibility
- **Style**: Clean, minimal, with clear visual hierarchy

## Page Structure

### 1. Header Section
- Full name "Dhruv Ozha" prominently displayed
- Professional titles: Frontend Developer | Full Stack Developer | UI/UX Designer
- Contact bar with email, GitHub, LinkedIn icons with links
- **PDF Download Button** in the top-right corner

### 2. Profile Summary
- Clean box with a brief career objective
- Highlights motivation, current studies, and career direction

### 3. Education Section
- B.E. in Computer Science Engineering - Swaminarayan University (1st Year)
- Additional training at Coding Gita Institute

### 4. Technical Skills Section
- Organized skill categories with clean badges/tags:
  - Frontend: HTML, CSS, JavaScript, React, Tailwind CSS
  - Backend: Node.js, Express.js, MongoDB
  - Languages: C, JavaScript
  - Tools: Git, GitHub, VS Code, Postman, etc.
  - Design: Figma
- Strongest skills (HTML/CSS, C) highlighted

### 5. Projects Section
- **Website Clone Project** with live link, description, tech stack, and role
- **Personal Portfolio** with details and tech stack
- Each project displayed in a clean card format with bullet points and action verbs

### 6. Social Links / Footer
- All social profiles (GitHub, LinkedIn, Instagram, Twitter/X, YouTube)
- Clean icon-based links
- Professional footer with email

- ✅ Print-optimized CSS for clean PDF output
